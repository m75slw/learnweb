## 类型注解
### 使用场景
- 声明变量时：变量所需要的变量类型

```ts
/**
let num:变量类型=变量值
**/
```

- 声明函数时：为函数参数和返回值指定类型
```ts
/**
function 函数名(变量名:形参类型):返回值类型 {
}
**/
```
- 声明接口时：可以规定属性和方法返回值的类型
```ts
/**
interface 接口名 {
    属性名: 类型,
    方法名: ()=>类型,
    属性名: 类型1|类型2|类型3|...
}
**/    
interface Name {
    first: string;
    second: string;
}
let name:Name;
name = {
    first: "Dwayne",
    second: "Johnson"
}
```
