
## Vue
**MVVM** 
- M：模型model数据
- V：视图view HTML
- VM：视图模型

**优势**
- 数据驱动：数据实现、页面渲染交给Vue，不操作DOM
- 组件化开发
- 虚拟DOM
- 指令

**场景**
- 应用场景：大量的数据交互`外卖、电商、后台管理系统`
- 不擅长：
  - DOM操作`动画多H5、企业站`
  - SEO差

**兼容性**  
> \>ie8

**配置项**
- el(string)：应用的范围或对应选择器
- data(对象)：定义应用数据
- directives：自定义局部指令


**创建**
1. html定义范围
2. 创建Vue实例
3. Vue实例里定义数据、方法
4. 数据、方法挂载到视图


- 注：
  - 数据和方法在范围内部
  - 模板中使用的数据或方法，一定要在实例中声明
  - 一个页面中可以有多个Vue实例，各自在固定的范围起作用
  - 实例中的数据，需要在创建实例之前进行定义
